# 🚀 TAGOS

## AI Powered Automated Content Tagging for Blogs

TAGOS is an AI-powered web application that automatically predicts relevant tags for blog articles using Natural Language Processing (NLP) and Deep Learning.

It uses DistilBERT to understand the textual meaning of a blog and generates the most relevant tags with confidence scores.

---

## ✨ Features

- 🧠 AI-powered blog tag prediction
- ⚡ DistilBERT-based NLP model
- 📊 Confidence score progress bars
- 📋 Copy predicted tags
- 📄 Download prediction report (PDF)
- 🔄 Reset functionality
- 🎉 Confetti animation after prediction
- 🌌 Aurora animated background
- 💎 Glassmorphism UI
- 📱 Responsive design
- 🚀 Fast Flask backend

Clone the repository

```bash

```

Go inside the project

```bash

```


Run the application

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```

## 🧠 How TAGOS Works

1. User enters a blog title.
2. User pastes the blog content.
3. DistilBERT converts the text into contextual embeddings.
4. The trained AI model predicts the most relevant tags.
5. Results are displayed with confidence scores.
6. Users can copy tags or download a PDF report.
